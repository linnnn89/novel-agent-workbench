## 2026-07-22：创作工作台 UI 第一阶段优化

### 关键变更

- 重构主窗口信息架构：左侧集中作品与章节，中间集中稿件编辑，右侧显示本章目标、上下文和模型服务。

- 新增不依赖 GUI 环境的桌面辅助函数测试。

- 未新增、升级或卸载依赖。
- 未改变草稿、确认稿、审稿、精修及模型调用的业务接口。
